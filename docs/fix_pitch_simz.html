fix_pitch_simz¶

One-off script to fix residual mismatches in cmd_states pitch and SIM-Z. These are largely due to SCS107 runs with a dense set of replans that are not exactly captured in the timeline loads.

This script uses telemetry to repair the errant command states. The result is still not perfect but mostly good. The last fix is 2007:153.

# Fetch comparison telemetry fetch –start 2002:010 –stop 2009:001:00:00:00 –dt 600 –outfile tlm2002_2008.dat –time-format secs aopcadmd cobsrqid tscpos aosares1 point_suncentang
